
Keystroke Logging Demonstration (Educational)
===========================================
IMPORTANT: This project is strictly for EDUCATIONAL PURPOSES.
- Use ONLY on your own system.
- Obtain explicit consent before any demonstration.
- Logs are stored locally and never transmitted.

Features:
- Starts logging after user confirmation.
- Stops logging when ESC is pressed.
- Saves logs to a local text file with timestamps.

Requirements:
- Python 3.8+
- Windows/Linux/macOS
- Library: pynput
